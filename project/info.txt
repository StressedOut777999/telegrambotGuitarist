from aiogram import Bot, Dispatcher, html:

Bot: Це клас, який представляє Telegram-бота. Він використовується для взаємодії з Telegram API, наприклад, для відправки повідомлень, фотографій, тощо.

Dispatcher: Це клас, який відповідає за обробку вхідних оновлень (наприклад, повідомлень, команд, callback-запитів). Він дозволяє реєструвати обробники подій.

html: Це модуль, який містить функції для форматування тексту у HTML-розмітці. Наприклад, html.bold("текст") створить жирний текст.

from aiogram.client.default import DefaultBotProperties:

DefaultBotProperties: Це клас, який дозволяє налаштувати властивості бота за замовчуванням, наприклад, режим парсингу повідомлень (HTML, Markdown тощо).

from aiogram.enums import ParseMode:

ParseMode: Це перерахування, яке визначає режим парсингу тексту. Наприклад, ParseMode.HTML дозволяє використовувати HTML-розмітку в повідомленнях, а ParseMode.MARKDOWN — Markdown.

from aiogram.filters import CommandStart, Command:

CommandStart: Це фільтр, який використовується для обробки команди /start. Він спрацьовує, коли користувач починає взаємодію з ботом.

Command: Це фільтр, який використовується для обробки інших команд. Наприклад, Command("help") спрацьовує, коли користувач вводить команду /help.

from aiogram.types import Message, CallbackQuery, URLInputFile:

Message: Це клас, який представляє повідомлення від користувача. Він містить інформацію про текст повідомлення, відправника, чат тощо.

CallbackQuery: Це клас, який представляє callback-запит, який надсилається, коли користувач натискає на кнопку в inline-клавіатурі.

URLInputFile: Це клас, який дозволяє завантажувати файли з інтернету за URL-адресою та відправляти їх у повідомленнях.
